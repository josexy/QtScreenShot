# QtScreenShot
QtScreenShot by Qt

### Compile

#### Git clone

```bash
git clone https://github.com/josexy/QtScreenShot.git --recursive
cd QtScreenShot
```

#### QHotkey

```bash
cd QHotkey
qmake
make
make install
cd ..
```

#### QtScreenShot

```bash
qmake
make
cd bin
./QtScreenShot
```
