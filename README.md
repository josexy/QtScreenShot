# QtScreenShot
QtScreenShot by Qt

### Compile

#### Git clone

```bash
git clone https://github.com/josexy/QtScreenShot.git --recursive
```

#### QHotkey

```bash
cd QtScreenShot/QHotkey
qmake ..
make
make install
cd ../..
```

#### QtScreenShot

```bash
mkdir build_qtscreenshot
cd build_qtscreenshot
qmake ..
make
./QtScreenShot
```
